# 🏋️‍♀️ GIRLS GYM - Fullstack Project

¡Bienvenida al repositorio oficial de **GIRLS GYM**! Este proyecto es una solución integral para la gestión de un gimnasio femenino, combinando una API potente y una interfaz moderna.

📍 **Ubicación:** Aguascalientes, México.
✨ **Estado:** En desarrollo (2026).

---

## 📂 Estructura del Proyecto

Este repositorio está dividido en dos partes principales:

1. **[Parte frontal del gimnasio](./Parte%20frontal%20del%20gimnasio):** Aplicación construida con **Angular 17**, enfocada en la experiencia de usuario, diseño neón y catálogo de máquinas.
2. **[Gym-API](./Gym-API):** Backend desarrollado en **Node.js** y **Express**, encargado de la conexión con **MongoDB** y la lógica de negocio.

---

## 🛠️ Tecnologías Utilizadas

### Frontend
- **Framework:** Angular
- **Estilos:** CSS3 con temática Dark/Neon
- **Estado:** Standalone Components

### Backend
- **Entorno:** Node.js
- **Base de Datos:** MongoDB (NoSQL)
- **Lógica:** Express.js

## 🚀 Cómo correr el proyecto
1. Clona el repositorio.
2. Entra a `Gym-API` y corre `npm start`.
3. Entra a `Parte frontal del gimnasio` y corre `ng serve`.
